## 交互说明
- 模型卡左右切换：`#modelPrev / #modelNext`
- FAQ 折叠展开：点击问题条目触发
- 免费试用提醒：
  - 12 秒后弹出右下提醒
  - 32 秒后弹出二次模态
  - 点击登录后停止提醒
- 帮助中心搜索：
  - 按分类筛选
  - 关键词搜索（支持 Enter）

## 接入你们官网项目的建议
1. 将新版内容拆成现有前端的 `Hero / Models / CoreFeatures / DownloadPricing / FAQ / TrialPopup` 组件。
2. 将帮助中心页接入 `help-center` 路由，并复用同一导航组件。
3. 保留你们现有埋点系统，在以下位置挂点：
   - 免费试用按钮点击
   - 模型轮播点击
   - 帮助中心搜索提交
   - 定价卡 CTA 点击
4. 将示例交互里的 `alert` 替换为真实登录、试用和支付流程。
