v1.20.1

2026.04.11

这一版继续强化多模型接入体验、下载入口整理和帮助中心内容组织，让新版首页、价格页和帮助中心之间的跳转更顺滑。

  • 统一顶部导航与下载下拉框交互。
  • 优化帮助中心入口、更新日志与搜索跳转体验。
  • 继续压缩页面层级，让常用入口更快触达。

v1.19.1

2026.02.27

图片生成工具支持 Nano Banana 2，进一步强化创意生成场景下的图像速度与质量，适合营销物料、概念草图和视觉探索任务。

  • 新增 Nano Banana 2 相关生成能力。
  • 支持更完整的图片生成工作流。
  • 提升创意类任务的连续输出效率。

v1.17.0

2025.10.27

自定义提供方增加 Gemini 和 Claude 风格的 API，便于在统一客户端中管理更多模型接入方式，减少来回切换配置的成本。

  • 自定义提供方支持更多 Gemini 风格接口。
  • 补充 Claude 风格 API 的接入配置。
  • 帮助自带 Key 用户更灵活地管理模型来源。

  • 新增 Gemini-2.5-flash-image 模型接入。
  • 适合更快的图片生成与版本试验。
  • 补充多模型图像生成的选择空间。
